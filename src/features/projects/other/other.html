Staż | Evalio

Projekt korporacyjny służący do okresowej ewaluacji pracowników, zbudowany w oparciu o architekturę Clean Architecture (Ardalis) przy użyciu Angular 20+ oraz ASP.NET Web API. Mój zakres prac:

  • Zastosowanie warstwowej architektury Clean Architecture, zapewniającej wysoką testowalność i łatwość utrzymania kodu.
  • Bezpieczna autentykacja użytkowników z wykorzystaniem MS Entra ID oraz biblioteki MSAL (silent sign-in).
  • Integracja z ekosystemem Microsoft: automatyczne powiadomienia e-mail oraz planowanie wydarzeń w kalendarzu Teams za pomocą procesów w tle (Background Jobs).
  • Wykorzystanie komponentów Kendo UI + Tailwind CSS, biblioteki AutoMapper oraz Entity Framework do zarządzania przepływem danych.

Inne

Miałam okazję uczestniczyć w wspólnym przebudowaniu strony opartej o Angular. Moim zadaniem było zadbanie o SEO strony:

  • Optymalizacja metadanych i treści pod kątem wyszukiwarek.
  • Dostosowanie formatu i wielkości zdjęć.
  • Zmiana na SSR.

Legacy

Posidam parę starszych projektów, które pomogły mi zdobyć doświadczenie w różnych technologiach i architekturach. Nie są one idealne, ale pokazują jaką drogę przebyłam w swojej naucę i pozwoliły mi zebrać cenne doświadczenie :)

InvoiceMaker (ASP.NET MVC)

Aplikacja webowa stworzona w technologii ASP.NET MVC (.NET 9) do kompleksowego zarządzania procesem fakturowania.

  • Implementacja wzorców CQRS oraz MediatR w celu separacji logiki odczytu i zapisu danych.
  • Automatyczne pobieranie danych kontrahentów z API Ministerstwa Finansów (MRiF) na podstawie numeru NIP.
  • Generowanie dokumentów PDF z wykorzystaniem biblioteki PDFsharp.
  • System autoryzacji i rejestracji oparty na ASP.NET Identity oraz asynchroniczne zarządzanie kontrahentami bez przeładowywania strony.
  • Komponenty Bootstrap 5.
  • 🔗 https://github.com/julitamajerx/InvioceMakerWeb - repozytorium

Blogger

Platforma blogowa oparta na ASP.NET 8 Razor Pages, oferująca pełen system zarządzania treścią.

  • Zarządzanie rolami i użytkownikami przy użyciu ASP.NET Identity.
  • Integracja z Cloudinary do przechowywania obrazów oraz edytorem Froala WYSIWYG do tworzenia treści postów.
  • System tagowania, komentowania i lajkowania treści przez zalogowanych użytkowników.
  • Komponenty Bootstrap 5.
  • 🔗https://github.com/julitamajerx/Blogger - repozytorium

InvoiceMaker Desktop

Aplikacja okienkowa Windows Forms (C#) stanowiąca fundament rozwoju systemów do fakturowania w moim portfolio.

  • Komunikacja z lokalną bazą danych przy użyciu LINQ to SQL.
  • Pobieranie danych z API MRiF oraz automatyczna numeracja faktur i przeliczanie kwot netto/brutto.
  • Generowanie i automatyczny zapis plików PDF do folderu Dokumenty użytkownika.
  • 🔗 https://github.com/julitamajerx/InvoiceMaker-WinForms - repozytorium
